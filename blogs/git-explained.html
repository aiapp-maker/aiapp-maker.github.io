Git in Action: Team Collaboration

Here's how teams use Git every day:

  1. Clone: Download the project to your computer
  2. Branch: Create your own workspace to make changes
  3. Commit: Save your changes with a description
  4. Push: Upload your changes to share with the team
  5. Pull Request: Ask the team to review and approve your changes
  6. Merge: Your changes become part of the main project
