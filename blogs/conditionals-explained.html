Comparison Operators

How we check conditions:

  • == (equals): age == 18
  • > (greater than): price > 100
  • < (less than): temperature < 0
  • >= (greater or equal): score >= 50
  • != (not equal): status != "active"

Combining Conditions

Check multiple things at once:

  • AND: IF age > 18 AND has_license THEN can drive
  • OR: IF weekend OR holiday THEN day off
